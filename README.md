# InfluFind
InfluFind adalah aplikasi katalog influencer yang membantu UMKM dan perusahaan menemukan kreator tepat untuk membangun brand awareness. Dengan tampilan sederhana dan warna lembut biru pastel serta krem, pengguna dapat menjelajahi profil influencer berdasarkan kategori tanpa login, sehingga memudahkan kolaborasi dan peningkatan visibilitas merek.
